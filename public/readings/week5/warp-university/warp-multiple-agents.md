# Warp에서 여러 에이전트 동시 사용하기

[영상 바로가기](https://www.youtube.com/watch?v=w0bJFC0u0pE)

## 핵심 요약

- **병렬 작업 관리**: Warp의 에이전트는 여러 작업을 동시에 처리하며, 데스크톱 알림으로 상태를 실시간 알려줌
- **음성 입력**: Whisper flow를 사용한 음성 모드로 타이핑 없이 자연스럽게 명령 입력
- **컨텍스트 자동완성**: @ context를 사용하여 파일 경로와 디렉토리를 자동완성하여 효율적으로 작업
- **블로킹 알림**: 에이전트가 코드 리뷰 등으로 블로킹되면 토스트 알림과 키보드 단축키로 즉시 확인 가능
- **자동 수락 모드**: 코드 diff를 처음 검토 후 올바른 방향이면 fast forward로 자동 수락 활성화
- **CLI 통합**: Warp는 cargo, git, gcloud 등 다양한 CLI를 인식하고 자동으로 적절한 명령어 선택
- **워크플로우 최적화**: 백로그 작업을 에이전트에 맡기고 중요한 제품 로드맵 작업에 집중 가능
