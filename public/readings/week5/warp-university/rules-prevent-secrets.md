# 규칙으로 비밀 유출 방지하기

[영상 바로가기](https://youtu.be/2ECPFKtQpVk)

## 핵심 요약

- 에이전트에게 코드 블록이나 명령어 요청 시 비밀을 공유하지 않도록 규칙 설정
- 페어 프로그래밍이나 라이브 스트리밍 시 특히 유용
- Warp의 비밀 삭제 기능을 활용하면 출력에서 비밀을 자동으로 숨김
- AI 에이전트가 민감한 정보를 보기 전에 사전 차단 가능

---

## 전체 번역

비밀을 보호하세요. 저는 에이전트가 코드 블록을 보여주거나 명령어를 요청할 때마다 비밀을 절대 공유하지 않도록 지시합니다. 이것은 제가 페어 프로그래밍을 하거나 작업을 라이브 스트리밍할 때 특히 유용합니다. 그리고 Warp를 사용한다면, 비밀 삭제(secret redaction) 기능을 켤 수 있습니다. 이 기능은 코딩 에이전트가 보기도 전에 출력에서 비밀을 자동으로 숨깁니다.
