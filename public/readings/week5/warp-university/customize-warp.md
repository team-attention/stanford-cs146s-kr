# Warp 커스터마이징

[영상 바로가기](https://youtu.be/fzb1JcZ0fFA)

## 핵심 요약

- **테마 커스터마이징**: Command P → "themes" 입력으로 테마 선택기 열기
- **입력창 위치 설정**: 세 가지 모드 - 상단에서 시작, 하단 고정, 상단 고정
- **AI 설정**: AI 기능 제어, 에이전트 자율성 수준 설정, 모델 선택(기본: Claude Sonnet)
- **권한 설정**: 코드 diff, 파일 읽기, 계획, 명령 실행에 대한 권한 레벨 조정 가능
- **코드베이스 인덱싱**: 코딩 시 Warp를 최대한 활용하기 위한 필수 설정
- **Warp Drive**: 팀과 프롬프트, 명령어, 노트북, 환경 변수 공유 가능
- **외관 설정**: 폰트, 앱 아이콘 등 다양한 시각적 요소 커스터마이징
- **키 바인딩**: Vim 모드를 포함한 키 바인딩 커스터마이징 가능
