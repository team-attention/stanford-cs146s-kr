## 번역 규칙

1. **원어 유지**: 업계에서 원어 그대로 사용하는 용어는 번역하지 않음
2. **음차 우선**: 적절한 한국어 번역이 없을 경우 음차 사용
3. **문맥 고려**: 같은 용어도 문맥에 따라 다르게 번역될 수 있음
4. **일관성**: 한 문서 내에서는 동일 용어에 동일 번역 사용

## 참고

- 새로운 용어가 등장하면 이 문서에 추가
- 기존 번역과 충돌 시 토론 후 결정
