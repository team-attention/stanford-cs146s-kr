# Warp vs Claude Code

[원본 링크](https://www.warp.dev/university/getting-started/warp-vs-claude-code)

## 개요

이 페이지에서는 Claude Code CLI와 Warp의 내장 코딩 에이전트를 비교하는 영상을 제공합니다.

## 비교 영상 내용

### 비교 항목

- **파일 작업** 및 코드 관리
- **코드 diff** 및 변경 사항 시각화
- **컨텍스트 수집** 기능
- **모델 선택** 옵션
- **설정** 구성

### 성능 비교

Sentry 이슈 테스트에서 두 도구 모두 비슷한 완료 시간(2~4분)을 보였으며, 각각 근본적인 문제를 성공적으로 파악하고 해결했습니다.

### 주요 차이점

**Claude Code의 강점:**
- 터미널 중심 워크플로우
- 서브 에이전트 기능

**Warp의 장점:**
- 내장 파일 트리
- diff 편집 기능
- GPT-5를 포함한 다중 모델 지원

## 참고

이 페이지는 상세한 문서 튜토리얼이나 코드 예제보다는 임베디드 비디오를 통한 시각적 비교에 중점을 둡니다.
