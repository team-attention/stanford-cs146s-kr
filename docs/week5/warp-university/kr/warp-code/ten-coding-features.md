# Warp에서 알아야 할 10가지 코딩 기능

[영상 바로가기](https://youtu.be/oeonZ-jtzhA)

## 전체 번역

이미 알고 계셨을 수도 있지만, Warp에서 직접 코드를 작성할 수 있습니다. 우리는 전통적인 개발 도구보다 코드베이스를 편집하고, 검토하고, 탐색하는 것을 훨씬 쉽게 만드는 기능 전체 세트를 구축했습니다. 시작하는 데 필요한 10가지 주요 기능을 보여드리겠습니다.

**첫 번째, 파일을 검색할 수 있습니다.**

Mac에서는 Command P, Windows나 Linux에서는 Ctrl+Shift+P로 명령 팔레트를 열 수 있습니다. 그런 다음 Mac에서는 Command O, Linux와 Windows에서는 Ctrl O를 눌러 파일 검색 모드를 활성화합니다. 이제 이 인터페이스에서 어떤 파일이든 검색하고 불러올 수 있습니다. 이렇게 하면 디렉토리로 cd 하거나 전체 경로를 기억할 필요 없이 프로젝트 내 파일 간을 이동하는 것이 매우 간단해집니다.

**두 번째는 탭 방식 파일 뷰어입니다.**

파일을 열면 Warp가 탭 뷰로 보여줍니다. 이는 화면 공간을 차지하는 분할 패널의 혼잡함을 피하면서도 여러 개의 열린 파일 간을 빠르게 전환할 수 있게 해줍니다.

**세 번째는 완전한 편집기 지원입니다.**

다른 현대적인 편집기에서 하는 것처럼 Warp 내에서 직접 코드를 타이핑하고 편집할 수 있습니다. 이를 위해 Vim 단축키를 외울 필요가 없습니다. 그냥 클릭하고, 텍스트를 강조 표시하고, 복사, 붙여넣기를 사용하거나 평소에 사용하던 일반적인 키보드 단축키를 사용하면 됩니다.

**네 번째는 찾기 및 바꾸기도 지원합니다.**

Warp의 편집기 경험은 정규표현식, 멀티커서 편집, 모두 바꾸기, 그리고 대소문자 보존 규칙까지 지원하여 함수와 변수 이름 변경이 빠르고 일관되게 이루어집니다.

**다섯 번째는 구문 강조입니다.**

Warp는 매우 다양한 언어와 프레임워크에 대한 구문 강조를 지원합니다. 에이전트가 생성한 코드 차이(diff)와 편집기 뷰에서 파일을 열 때 모두 나타나서, 파일을 시각적으로 훨씬 쉽게 스캔할 수 있게 만듭니다.

**여섯 번째는 참조된 파일로의 원클릭 링크입니다.**

Warp의 에이전트 모드가 파일을 참조할 때, 해당 코드의 정확한 줄로 직접 링크됩니다. 그것을 클릭하면 Warp에서 해당 파일의 정확한 코드 줄로 열 수 있습니다. 이러한 투명성은 에이전트가 무엇을 하고 있는지 추적하기 쉽게 만들어줍니다. 파일을 읽든, 로직을 추적하든, 수정을 하든 말이죠. 설정의 기능 섹션에서 Warp를 편집기로 설정해야 이러한 파일 링크가 Warp 코딩 뷰 내에서 직접 열린다는 점을 참고하세요.

**일곱 번째는 코드 리뷰 패널입니다.**

Warp는 이제 에이전트가 수정한 모든 파일과 차이(diff)를 요약하는 전용 패널을 제공합니다. 한눈에 검토하고, 변경사항을 승인하거나 편집할 수 있으며, 심지어 다음 프롬프트에서 이러한 차이를 직접 참조할 수도 있습니다. 이는 환각(hallucination)을 줄일 뿐만 아니라 에이전트를 실제 코드에 기반하도록 유지합니다.

**여덟 번째는 코드 스니펫 참조입니다.**

Warp는 코드베이스에 대해 무언가를 설명할 때 코드 블록을 표면화합니다. 저는 이 기능이 좋은데, 다음 요청을 위한 새로운 컨텍스트로 해당 스니펫을 첨부할 수 있기 때문입니다. 이렇게 하면 전체 히스토리를 메모리에 유지할 필요가 없습니다. 중요한 코드만 전달할 수 있어서 토큰 사용을 최소화하고 에이전트의 집중력을 날카롭게 유지합니다.

**아홉 번째는 코드베이스 인덱싱입니다.**

Warp는 리포지토리를 자동으로 인덱싱할 수 있으며, 이는 Warp가 이미 프로젝트 구조를 이해하고 있기 때문에 에이전트가 대규모 코드베이스를 요약하거나, 버그를 수정하거나, 리팩토링을 처리하는 것을 극적으로 빠르게 만듭니다.

**마지막으로 파일 트리 뷰를 추가했습니다.**

이 아이콘을 클릭하면 전체 리포지토리를 탐색하고 필요한 파일을 열 수 있습니다.

## 핵심 요약

- **파일 검색**: Command/Ctrl+O로 빠르게 파일 검색 및 이동
- **탭 뷰어**: 화면을 효율적으로 사용하는 탭 방식 파일 관리
- **완전한 편집기**: Vim 없이도 사용 가능한 현대적 코드 편집 기능
- **찾기/바꾸기**: 정규표현식, 멀티커서, 대소문자 보존 지원
- **구문 강조**: 다양한 언어 지원으로 코드 가독성 향상
- **파일 링크**: 에이전트가 참조한 코드 라인으로 즉시 이동
- **코드 리뷰 패널**: 에이전트 변경사항을 한눈에 검토
- **스니펫 참조**: 필요한 코드만 컨텍스트로 전달하여 효율성 향상
- **코드베이스 인덱싱**: 자동 인덱싱으로 대규모 프로젝트 빠른 처리
- **파일 트리 뷰**: 리포지토리 전체를 시각적으로 탐색
