## 핵심 요약

- Warp의 멀티스레딩 기능으로 **여러 에이전트 탭에서 동시에 작업** 가능
- **3가지 작업 동시 진행**: UI 코드 수정, PR 리뷰 현황 확인, 로그 분석
- **음성 입력 지원**: 긴 프롬프트를 타이핑 대신 말로 입력하여 효율성 증가
- **MCP 통합**: Linear 이슈를 자동으로 읽어와 컨텍스트 파악
- **저장된 프롬프트 재사용**: 팀 전체에서 워크플로우 공유 가능
- **실시간 작업 모니터링**: 사이드 패널에서 모든 실행 중인 작업 한눈에 확인
- **실용적인 사례**: 포커스되지 않은 입력창의 배경 제거 기능을 자동으로 구현하고 즉시 테스트
