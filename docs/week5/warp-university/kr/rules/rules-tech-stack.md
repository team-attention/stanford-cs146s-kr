# 규칙으로 기술 스택 선호도 설정하기

[영상 바로가기](https://youtu.be/W5B6MhZsZ_4)

## 전체 자막

당신이 사용하고 싶은 기술을 적어두세요. 웹 앱을 만들어달라고 요청할 수는 있지만, 에이전트는 5년 전의 React와 Express 관행을 사용할 것입니다. 저는 웹사이트에는 Astro를 사용하고 데스크톱 앱에는 SvelteKit을 사용한다고 말하고 싶습니다. 그러면 에이전트가 다음번에 프로젝트를 구축할 때 그 선호도를 기억할 것입니다.

---

## 핵심 요약

- AI 에이전트는 기본적으로 오래된 기술 스택과 관행을 사용할 수 있음
- 규칙을 통해 선호하는 기술 스택을 명시적으로 지정해야 함
- 예시: "웹사이트는 Astro, 데스크톱 앱은 SvelteKit 사용"
- 에이전트는 한 번 설정한 기술 선호도를 기억하고 재사용함
- 프로젝트 스캐폴딩 시 일관된 기술 스택 적용 가능
