# Warp UI 미니멀하게 만들기

[영상 바로가기](https://youtu.be/1GKsIT8FSsE)

## 전체 자막

많은 분들이 가끔 Warp의 UI가 복잡해 보이거나 무겁게 느껴진다고 말씀하셨는데, 충분히 이해합니다. 때로는 그 단순한 CLI 느낌이 필요할 때가 있죠. 그래서 Warp의 UI를 훨씬 더 미니멀하게 만드는 몇 가지 팁을 소개합니다. 가장 쉬운 방법은 Mac에서는 Command + P, Windows와 Linux에서는 Control + Shift + P로 명령 팔레트를 여는 것입니다. 거기서 "disable"을 입력하기 시작하면 다음과 같은 옵션들이 보입니다. "disable auto suggestions"로 인라인 제안을 끄고, "disable block dividers"로 명령어 사이의 구분선을 제거하고, "disable voice input"으로 마이크 아이콘을 숨기고, "disable active AI"로 여기 이 고스트 프롬프트가 나타나지 않게 하거나, "disable completion"으로 기본적으로 완성 메뉴가 뜨지 않게 할 수 있습니다. Vim 단축키로 입력을 편집할 필요가 없다면 "disable Vim status bar"로 이 작은 아이콘도 숨길 수 있습니다. "disable tab indicators"로 각 탭 세션에서 무슨 일이 일어나고 있는지 알려주는 이 컬러 탭 마커들을 숨길 수도 있고요. "disable dimming inactive pane"으로 분할 창 UI가 이렇게가 아니라 이렇게 보이도록 만들 수 있습니다. 명령 팔레트에서 "disable"을 입력하고 뭐가 나오는지 보면 아마 더 많은 설정들을 발견할 수 있을 겁니다. 하지만 이것만 해도 Warp가 많이 깔끔해질 것입니다. 테마도 바꿔볼 수 있습니다. UI가 매우 복잡하고 지저분하게 느껴지게 만드는 테마들이 많이 있거든요. 저는 개인적으로 단순하고 차분한 회색 톤을 가진 Afterbury 테마를 좋아합니다. 더 클래식한 느낌을 원한다면 유니버설 프롬프트 대신 클래식 프롬프트로 전환하는 것을 추천합니다.

유니버설 프롬프트는 AI 기능을 위해 만들어졌습니다. 그래서 슬래시 명령어, 음성 입력, 이미지, 컨텍스트, 에이전트 모드 같은 것들을 제공하죠. 하지만 그런 게 필요하지 않다면, 클래식 프롬프트가 매우 간결한 터미널 느낌을 줍니다.

프롬프트 커스터마이저에 들어가서 어떤 칩들이 나타날지 정확히 조정할 수도 있습니다. 단순한 파일 경로만 원한다면 그렇게 할 수 있죠.

또 하나의 작은 조정은 Warp가 호버할 때만 탭 바를 보여주도록 설정하는 것입니다. 이렇게 하면 정말 필요할 때까지 탭들이 시야에서 사라집니다. 그래도 Warp가 너무 복잡하게 느껴진다면, AI를 완전히 비활성화할 수도 있습니다. 이렇게 하면 AI 워크플로우를 지원하기 위해 존재하는 많은 추가 UI 요소들이 제거됩니다. 기본적으로 Warp를 AI도 없고 군더더기도 없는 깔끔한 터미널 경험으로 되돌리는 것이죠. 물론 AI 기능은 잃게 되겠지만, 무엇보다 속도와 단순함을 찾고 있다면 이것이 완벽한 설정일 수 있습니다.

## 핵심 요약

- **명령 팔레트로 빠른 설정**: `Command + P` (Mac) 또는 `Control + Shift + P` (Windows/Linux)에서 "disable" 검색으로 다양한 UI 요소 비활성화
- **주요 비활성화 옵션**: 자동 제안, 블록 구분선, 음성 입력, AI, 완성 메뉴, Vim 상태바, 탭 표시기, 비활성 창 어둡게 하기
- **테마 변경**: Afterbury 같은 미니멀한 테마로 전환하여 깔끔한 느낌 유지
- **클래식 프롬프트**: AI 기능이 필요 없다면 유니버설 프롬프트 대신 클래식 프롬프트 사용으로 간결한 터미널 느낌 제공
- **프롬프트 커스터마이징**: 프롬프트 커스터마이저에서 필요한 요소만 표시하도록 조정 가능
- **탭 바 자동 숨김**: 호버 시에만 탭 바 표시하도록 설정하여 화면 정리
- **AI 완전 비활성화**: 최대한 깔끔한 경험을 원한다면 AI 기능 전체를 비활성화하여 순수 터미널 경험 구현
