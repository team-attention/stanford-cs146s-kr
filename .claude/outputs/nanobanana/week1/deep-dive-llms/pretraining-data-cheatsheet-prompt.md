### Output Style: "Nano Banana" Cheat Sheet

다음 포맷팅 규칙을 엄격히 따라주세요:

1. **시각적 계층 구조**:
   - 엄격한 Markdown 구조 사용
   - 각 섹션 헤더에 구체적인 이모지 사용
   - **굵은 글씨**로 핵심 개념 및 정의 강조
   - 타임라인이 아닌 주제별 그룹화

2. **다이어그램 & 도식 (중요)**:
   - `mermaid` 코드 블록으로 개념 시각화
   - 전체 주제 구조를 **Mind Map**으로 생성
   - 데이터 처리 과정을 **Flowchart**로 생성
   - 데이터 유형/단계 관계를 **표**로 정리

3. **개념 테이블**:
   - 핵심 개념을 표 형식으로 정의와 함께 정리
   - 처리 단계별 비교 테이블 생성

4. **핵심 인사이트**:
   - 강의에서 기억할 만한 인용구나 핵심 통찰 추출
   - 데이터 규모에 대한 직관적 이해 포인트 강조
